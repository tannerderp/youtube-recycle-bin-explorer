# Youtube Recycle Bin Explorer

Use this sick website to explore the depths of youtube and see a multitude of low viewed videos. Try it out at the github pages link [here](https://tannerderp.github.io/youtube-recycle-bin-explorer/)!

## Random Recommendations
- Log out of youtube in order to stop reccomendations from throwing off the searches
- Enter in your own dates to searches with dates in them (such as WIN YYYYMMDD) to see videos from that day in history
